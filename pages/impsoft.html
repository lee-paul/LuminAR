Software Used

To create the Unity Plugins we used a variety of software.

The Unity 3D Platform comes with a customised version of the MonoDevelop IDE, MonoDevelop has similar features to those of NetBeans and Microsoft Visual Studio.

Augmented Reality is made possible by using the Vuforia Software Platform, the Vuforia SDK can be easily downloaded and implemented in to Unity by downloading the Vuforia-Unity package.

The Java (.jar and .java) files are created in the Eclipse IDE. The Java files are necessary to access the Android features needed for the GPS plugins. A simple ANT script is used to create the .jar (plugin) file.
