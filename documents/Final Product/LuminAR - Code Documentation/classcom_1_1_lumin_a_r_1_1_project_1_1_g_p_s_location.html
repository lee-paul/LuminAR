static void com.LuminAR.Project.GPSLocation.startLocationListeners ( )
static

Starts the GPS stuff

This function starts the location listener. The location listenser is used to find the location of the user. The location is found by accessing the device's GPS hardware and tracks a location.

The function also uses a network location listener. The network location listener is used to find the device's location using the network of the device. The location is set by the network and is not as accurate as the device's GPS sensor hardware.
