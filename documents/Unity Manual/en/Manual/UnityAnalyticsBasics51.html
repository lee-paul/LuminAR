Basic Integration

The Basic Integration steps will walk you through how to add your unique ProjectId to your game that triggers the user data collection process. Once you add this ProjectId, Unity Analytics will automatically track user engagement and behavioral data for the entire duration of the user’s play session.

This Basic Integration reference contains conceptual overviews and samples to help you begin using Unity Analytics. Basic Integration is quick, easy and should only take ~5 minutes to complete.
